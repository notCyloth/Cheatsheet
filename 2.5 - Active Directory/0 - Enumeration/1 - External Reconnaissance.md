# What are we looking for?
## IP Addresses/Autonomous System Number (ASN)
- Cloud hosting providers
- DNS Entries
## Domain Information
- Who administers domain
- Subdomains
- Mailservers
- DNS
- Websites
- VPN Portals
## Schema Format
- Email schema
- AD Usernames
- Password policies
## Data Disclosures
- pdf, ppt, docx, xlsx
- Intranet site listings
- User metadata
- Shares
## Data Breaches
For usernames, passwords, etc.
